con los diseños contenidos multimediales e interactivos, basados en HTML5, CSS3 y otras tecnologias, se añaden diferentes etiquetas para el aprendizaje el principal objetivo de esta Web es entender el funcionamiento de los contenedores Flex Box para que sirven y como utilizarlos, tambien aprenderemos como crear Listas y enlaces en HTML.

Flex Box

el posicionamiento flexible o flexbox permite acomodar los elementos de una página según cambien las dimensiones y orientación de la página. la idea principal es que un contenedor flexible expande o comprima sus elementos para rellenar el espacio libre o ajustarse al area disponible.


para mostrar las caracteristicas del flexbox se debe crear el contenedor con el siguiente codigo.
div class="flex-container"
div
container1 /div div
container2 /div div
container3 /div div
container4 /div
/div



tambien debemos utilizar una hoja de estilos CSS para darle formato a los elementos

.flex-container{
display: flex;
background-color: burlywood;
padding: 10px;
border: 2px solid black;
}

.flex-container div{
background-color: gainsboro;
width: 100px;
magin: 10px;
text-alingn: center;
line-height: 75px;
font-size: 30px;
border: 2px solid black;
}



los flexbox manejan direccion con la propiedad flex-direction el cual especifica la direccion principal de como colocar los elementos en el contenedor. los valores que se utilizan son: row(fila), row-reverse (fila invertida), column(columna), column-reverse(columna.invertida).

#container1{
flex-direction: row;
}br #container2{
flex-direction: row-reverse;
}br #container3{
flex-direction: column;
}
#container4{
flex-direction: column-reverse;
}




hay otras propiedades que nos permiten ordenarlos y que ocupen un espacio estas propiedades son:
order: este indica el orden donde quiere que aparezca el elemento dentro del contenedor dentro del div hijo se coloca el siguiente codigo: order: 1
flex-grow: esta propiedad establece cuanto crecera el elemento en relacion con los otros y lo indicamos asi: flex-grow: 2
flex-sharing: es lo contrario se encoge de tamaño en relacion con los otros.
flex-basis: indica el tamaño por defecto (ancho) del elemento antes que el espacio libre sea distribuido normalmente se deja el valor por defecto flex-basis: auto.

Listas y enlaces en HTML

En HTML5 se definen 3 tipos listas en las cuales las voy a listar como ejemplo y las listas tambien se pueden anidar unas dentro de otras

  1. listas con numeros
  2. listas con viñetas
  3. listas con definiciones
  • listas numeradas: sirven para crear listas con numeros. y etiqueta es ol /ol y
  • listas con viñetas: estas sirven para crear diferentes listas con simbolos y viñetas y la etiqueta ul /ul
  • listas de definiciones: sirve para definir terminos y la etiquetas dl /dl.

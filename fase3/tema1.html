Flex Box


para mostrar las caracteristicas del flexbox se debe crear el contenedor con el siguiente codigo.
div class="flex-container"
div
container1 /div div
container2 /div div
container3 /div div
container4 /div
/div



tambien debemos utilizar una hoja de estilos CSS para darle formato a los elementos

.flex-container{
display: flex;
background-color: burlywood;
padding: 10px;
border: 2px solid black;
}

.flex-container div{
background-color: gainsboro;
width: 100px;
magin: 10px;
text-alingn: center;
line-height: 75px;
font-size: 30px;
border: 2px solid black;
}



los flexbox manejan direccion con la propiedad flex-direction el cual especifica la direccion principal de como colocar los elementos en el contenedor. los valores que se utilizan son: row(fila), row-reverse (fila invertida), column(columna), column-reverse(columna.invertida).

#container1{
flex-direction: row;
}br #container2{
flex-direction: row-reverse;
}br #container3{
flex-direction: column;
}
#container4{
flex-direction: column-reverse;
}




hay otras propiedades que nos permiten ordenarlos y que ocupen un espacio estas propiedades son:
order: este indica el orden donde quiere que aparezca el elemento dentro del contenedor dentro del div hijo se coloca el siguiente codigo: order: 1
flex-grow: esta propiedad establece cuanto crecera el elemento en relacion con los otros y lo indicamos asi: flex-grow: 2
flex-sharing: es lo contrario se encoge de tamaño en relacion con los otros.
flex-basis: indica el tamaño por defecto (ancho) del elemento antes que el espacio libre sea distribuido normalmente se deja el valor por defecto flex-basis: auto.

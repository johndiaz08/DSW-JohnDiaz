Listas y enlaces en HTML

En HTML5 se definen 3 tipos listas en las cuales las voy a listar como ejemplo y las listas tambien se pueden anidar unas dentro de otras

    listas con numeros listas con viñetas listas con definiciones
  • listas numeradas: sirven para crear listas con numeros. y etiqueta es ol /ol y
  • listas con viñetas: estas sirven para crear diferentes listas con simbolos y viñetas y la etiqueta ul /ul
  • listas de definiciones: sirve para definir terminos y la etiquetas dl /dl.

    • Los enlaces o tambien conocidos como hipervinculos o hiperenlaces que nos permiten mover entre diferentes partes del documento.
    para añadir un enlace usamos la etiqueta
    a href="">
    donde estan las comillas se coloca el destino que puede ser:

      una direccion absoluta: http://www.unad.com.co
      una direccion relativa: fase2/design.html
      un correo electronico: mailto:jhondiaz08@gmail.com
